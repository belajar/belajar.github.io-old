Memulakan pengaturcaraan dalam JavaScript

2015-03-30

Memulakan pengaturcaraan dalam JavaScript agak mudah kerana boleh dikatakan semua pelayar web (browser) utama seperti Google Chrome, Mozilla Firefox, Internet Explorer atau Apple Safari mempunyai sokongan JavaScript. Jadi jika anda mempunyai pelayar web di atas, anda boleh mula menulis kod JavaScript serta-merta.

JavaScript hari ini bagaimanapun tidak lagi terhad untuk dijalankan pada browser sahaja. Ia juga turut digunakan pada server-side melalui teknologi seperti NodeJS dan sebagainya.
